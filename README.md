# 펫 프로그램 V1.0
- 캐릭터가 모니터 화면을 좌/우 움직이면서 돌아다니는 프로그램임

## 프로젝트 개요
본 프로젝트는 단순 캐릭터가 모니터를 활보하는 프로그램입니다.
객체지향인 Python를 공부하고 난 다음에 한 번 활용을 하고 싶어서 뭘 만들지 고민하다가 대학 동기가 펫 프로그램을 만들어 보라고해서 만들게 되었습니다

## 회고 및 느낀 점
- **도전 과제**: 객체지향이 아직 미숙한 상태에서 PySide6까지 처음 다뤄, 만드는 과정에 어려움이 많았다.
- **해결 과정**: 그런 이유로 생성형 AI에게 완성 코드가 아닌 힌트만 받아가며 직접 구현해 완성했다.
- **느낌 점**: 만들면서 객체지향과 PySide6을 어떻게 다뤄야 하는지 어느 정도 감을 잡았다. 다만 AI가 힌트가 아닌 온전한 답을 줄 때가 있었고, 스스로 완전히 이해하고 넘어간 게 아니라서 현재 실력에 대한 부족함을 느꼈다.

## 주요 기능
- **동작**: 캐릭터가 모니터 화면을 좌/우 배회하거나 쉬는 모션을 행동한다
- **애니메이션**: 걷기, 제자리에서 쉬기, 시작 모션, 종료 모션, 가만히 있기가 있다.
- **배경화면**: 배경화면이 투명하게 처리되어 있어 다른 작업을 해도 문제가 없다.
- **상호작용**: 지금은 설정창에 종료 버튼만 상호작용을 하고 있다. (나중에 시간이 되면 다양한 모션을 추가할 예정)

## 설치 및 시작하기
**Python이 설치된 경우**
1. 저장소 소스 파일 다운로드
2. main.pyw 실행

**Python이 없는 경우**
1. Tag(Releases)에서 .exe 다운로드
2. 실행
