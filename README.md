# 펫 프로그램 V1.0
- 캐릭터가 모니터 화면을 좌/우 움직이면서 돌아다니는 프로그램임

## 프로젝트 개요
본 프로젝트는 단순 캐릭터가 모니터를 활보하는 프로그램입니다.
객체지향인 Python를 공부하고 난 다음에 한 번 활용을 하고 싶어서 뭘 만들지 고민하다가 대학 동기가 펫 프로그램을 만들어 보라고해서 만들게 되었습니다

## 회고 및 느낀 점
- **도전 과제**: 아직 객체지향이 미숙함과 Pyside6을 처음 활용하기 위해 만드는데 많은 문제가 있었다
- **해결 과정**: 그러면 이유로 생성형 AI를 통해 힌트만 보고 제작을 하여 완성했다.
- **느낌 점**: 그래도 만들면서 객체지향 및 Pyside6을 어떤게 다뤄어야 하는지는 어느정도는 이해했다. 다만, 아쉬운 점은 생성형 AI가 힌트가 아닌 온전한 답을 보여주는 경우와 내가 완전한 이해를 만들고 한 것이 아니여서 자신의 현 상태에 대한 심각함을 느꼈다.

## 주요 기능
- **동작**: 캐릭터가 모니터 화면을 좌/우 배회하거나 쉬는 모션을 행동한다
- **애니메이션**: 걷기, 제자리에서 쉬기, 시작 모션, 종료 모션, 가만히 있기가 있다.
- **배경화면**: 배경화면이 투명하게 처리되어 있어 다른 작업을 해도 문제가 없다.
- **상호작용**: 지금은 설정창에 종료 버튼만 상호작용을 하고 있다. (나중에 시간이 되면 다양한 모션을 추가할 예정)

## 설치 및 시작하기
- Python이 설치된 경우 저장소에 있는 소스 파일 다운로드 후 main.pyw를 실행
- 설치가 되어 있지 않은 경우 Tag에 .exe를 다운로드 해서 실행 (이 프로그램은 안전하니 보안 문제가 떠도 그냥 실행하셈)
